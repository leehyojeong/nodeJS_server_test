##개인 웹 포트폴리오(구현중)
- NodeJS, Express를 사용함
- DB는 MongoDB를 사용하여 프로젝트 리스트를 관리할 예정
- 시작 페이지, 소개 페이지, 스킬 페이지, 컨택트 페이지 디자인은 완성
- 애니메이션 더 추가 예정
- 프로젝트 페이지는 아직 구상중
